# Bosnian News and Comments Analytics
## Supported portals:
- Klix
- Faktor
- Dnevni avaz

## Technologies used:

- Python, Django
- JavaScript, HTML, CSS, Materializecss, Chart js
- BeautifulSoup, Selenium

## Sentiment analysis algorithm:
- Bag of Words
